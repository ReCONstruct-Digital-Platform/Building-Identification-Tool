Q{{form.field_ordering.self_similar_cluster}}: Self-Similar Clusters

Definition:

HAHAHAHAH A building is part of a self-similar cluster if it close to other buildings built at the same time, following the same construction methods, and using the same materials. Most of the time, they will seem virtually identical copies.

Importance:

This is important for retrofits as a good candidate building part of a cluster creates economies of scale which makes retrofitting the whole cluster more attractive than retrofitting a single building.

Note:

Presence of the Multiple Addresses tag makes this more likely as the evaluation unit could contain multiple buildings.

Example 1:

We can see from the streetview that there is an almost identical building to the right. The satellite view gives a bigger picture where wee see that there seems to be a 5 building cluster.

Example 2:

Again, it is clear from the streetview image that there are multiple similar buildings. The satellite view shows a cluster of 4 (or 8) buildings.
